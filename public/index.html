Akun Google — simpan & sinkronkan pengaturan

Login Google menyimpan bot token, chat ID, Helius key, dan watchlist kamu ke Google Drive milikmu sendiri (folder tersembunyi khusus aplikasi ini) — otomatis tersinkron di semua perangkat/browser. Tanpa login, data hanya tersimpan di browser ini.

Telegram — kirim signal

Signal watchlist (dev launch token baru, lengkap dengan CA) dikirim otomatis selama halaman ini terbuka; signal hasil scan dikirim lewat tombol "Kirim signal Telegram" setelah scan selesai. Token & key tersimpan hanya di browser kamu (localStorage), tidak pernah disimpan di server.

Helius — dukungan lintas launchpad

Tanpa key: analisis hanya pump.fun + trace on-chain dasar. Dengan Helius (gratis): paste CA Meteora/launchpad lain tetap ketemu dev-nya, plus tombol "Scan deploy lintas launchpad" di profil dev untuk melihat SEMUA token yang pernah dia deploy, launchpad apa pun.

Cara mendapatkan token & chat ID

  1. Buka Telegram, chat @BotFather, kirim /newbot, ikuti instruksinya — kamu dapat bot token.
  2. Chat @userinfobot untuk melihat chat ID kamu (angka).
  3. Buka chat dengan bot barumu dan tekan Start (wajib, supaya bot boleh mengirimimu pesan).
  4. Tempel token & chat ID di sini, Simpan, lalu "Kirim pesan tes".
