Auto-buy bot — beli otomatis saat dev watchlist launch

Bot lokal mendengar launch pump.fun real-time (WebSocket, delay ~1–3 detik) dan bertindak saat creator-nya ada di watchlist kamu. Atur di sini, lalu jalankan node bot.js di folder ini. Bot berjalan di komputermu — private key tidak pernah menyentuh website.

Panel ini hanya berfungsi saat website dibuka secara lokal (localhost). Auto-buy tidak bisa jalan di versi Vercel karena butuh private key di mesinmu.

Parameter beli

Auto-sell — amankan modal & ambil profit

Bot memantau harga token yang dibeli secara real-time dan menjual otomatis saat mencapai target. Contoh: jual 50% di 2x = balik modal, sisanya jalan gratis.

Take-profit 1 jual % di x
Take-profit 2 jual % di x (0 = nonaktif)
Stop-loss jual % di x (0 = nonaktif)

Take-profit berlaku di simulasi maupun live — di simulasi kamu dapat sinyal jual tanpa eksekusi, jadi bisa uji strateginya dulu.

Wallet burner (hanya untuk mode Live)

Pakai wallet khusus bot berisi dana kecil — bukan wallet utamamu. Private key format base58 (ekspor dari Phantom: Settings → Export Private Key). Tersimpan hanya di bot-config.json di komputermu, tidak pernah dikirim ke server mana pun.

Cara menjalankan

  1. Isi parameter di atas → Simpan & sinkron ke bot (menulis bot-config.json).
  2. Buka terminal di folder ini, jalankan node bot.js. Biarkan jendela terbuka — selama itu bot memantau.
  3. Mode Simulasi dulu beberapa hari: setiap dev watchlist launch, kamu dapat sinyal Telegram tanpa uang. Cek akurasinya.
  4. Kalau sudah yakin, pindah ke Live + isi private key wallet burner, Simpan lagi, restart bot. Untuk 24/7 taruh di VPS.

Telegram — kirim signal

Signal watchlist (dev launch token baru, lengkap dengan CA) dikirim otomatis selama halaman ini terbuka; signal hasil scan dikirim lewat tombol "Kirim signal Telegram" setelah scan selesai. Token & key tersimpan hanya di browser kamu (localStorage), tidak pernah disimpan di server.

Helius — dukungan lintas launchpad

Tanpa key: analisis hanya pump.fun + trace on-chain dasar. Dengan Helius (gratis): paste CA Meteora/launchpad lain tetap ketemu dev-nya, plus tombol "Scan deploy lintas launchpad" di profil dev untuk melihat SEMUA token yang pernah dia deploy, launchpad apa pun.

Cara mendapatkan token & chat ID

  1. Buka Telegram, chat @BotFather, kirim /newbot, ikuti instruksinya — kamu dapat bot token.
  2. Chat @userinfobot untuk melihat chat ID kamu (angka).
  3. Buka chat dengan bot barumu dan tekan Start (wajib, supaya bot boleh mengirimimu pesan).
  4. Tempel token & chat ID di sini, Simpan, lalu "Kirim pesan tes".
