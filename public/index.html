Auto-buy bot — buy automatically when a watchlist dev launches

A local bot listens to pump.fun launches in real time (WebSocket, ~1–3s delay) and acts when the creator is on your watchlist. Configure it here, then run node bot.js in this folder. The bot runs on your machine — your private key never touches the website.

This panel only works when the website is opened locally (localhost). Auto-buy can't run on the Vercel version because it needs a private key on your machine.

Burner wallet (Live mode only)

Use a dedicated bot wallet with a small amount — not your main wallet. Private key in base58 (export from Phantom: Settings → Export Private Key). Stored only in bot-config.json on your machine, never sent to any server.

How to run

  1. Fill in the parameters above → Save & sync to bot (writes bot-config.json).
  2. Open a terminal in this folder and run node bot.js. Keep the window open — the bot monitors while it's running.
  3. Run Simulation mode for a few days first: every time a watchlist dev launches, you get a Telegram signal with no money involved. Check the accuracy.
  4. Once you're confident, switch to Live + add your burner wallet private key, save again, restart the bot. For 24/7, put it on a VPS.

Telegram — send signals

Watchlist signals (a dev launching a new token, with the CA) are sent automatically while this page is open; scan signals are sent via the "Send Telegram signal" button after a scan finishes. Token & key are stored only in your browser (localStorage), never on a server.

Helius — cross-launchpad support

Without a key: analysis is pump.fun only + basic on-chain trace. With Helius (free): paste a Meteora / other launchpad CA and still find the dev, plus a "Scan cross-launchpad deploys" button on the dev profile to see EVERY token they've ever deployed, on any launchpad.

How to get a token & chat ID

  1. Open Telegram, chat @BotFather, send /newbot, follow the steps — you get a bot token.
  2. Chat @userinfobot to see your chat ID (a number).
  3. Open a chat with your new bot and press Start (required, so the bot is allowed to message you).
  4. Paste the token & chat ID here, Save, then "Send test message".
